# 实现计划

## 1. 安装依赖库
- 分析 `booking.py` 文件，确定所需的第三方库
- 主要依赖：`selenium`
- 使用 `pip install selenium` 安装

## 2. 开发自动更新 chromedriver 脚本
创建 `update_chromedriver.py` 脚本，实现以下功能：
- 获取本地 Chrome 浏览器版本
- 根据 Chrome 版本下载对应的 chromedriver.exe
- 替换当前文件夹中的 chromedriver.exe

### 实现步骤：
1. 使用 `subprocess` 调用 `chrome --version` 获取本地 Chrome 版本
2. 解析版本号，提取主版本号（如 120.0.6099.109 提取为 120）
3. 从 chromedriver 官方下载链接获取对应版本的 chromedriver
4. 下载并解压 chromedriver.exe
5. 替换当前文件夹中的 chromedriver.exe

## 3. 测试脚本
- 运行 `update_chromedriver.py` 验证是否能正确更新 chromedriver
- 运行 `booking.py` 验证是否能正常使用更新后的 chromedriver
